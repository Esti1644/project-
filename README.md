## 🐛 פתרון בעיות

**❌ לא מצליחה להתחבר?**
→ פתחי `test.html` ולחצי "אפס והתחל מחדש"

**❌ התמונות לא נטענות?**
→ בדקי את הקישורים ב-storage.js

**❌ ההקראה לא עובדת?**
→ השתמשי בדפדפן Chrome (מומלץ ביותר)

**❌ האתר לא נראה טוב?**
→ נקי Cache: Ctrl+Shift+Delete

## 💡 טיפים מקצועיים

## 🌐 העלאה לאינטרנט - אפשרויות חינמיות

### 1️⃣ GitHub Pages (מומלץ ביותר!) ⭐
```bash
1. צרי חשבון ב-GitHub.com
2. צרי repository חדש
3. העלי את כל הקבצים מתיקיית recipe-app
4. Settings → Pages → הפעילי GitHub Pages
5. הקישור: https://username.github.io/repository-name
```

### 2️⃣ Netlify (הכי קל!)
1. היכנסי ל-netlify.com
2. גררי את תיקיית recipe-app לאתר
3. קבלי קישור מיידי! 🎉

### 3️⃣ Vercel (מהיר ומקצועי)
1. היכנסי ל-vercel.com
2. העלי את הפרויקט
3. קבלי קישור אוטומטי

### 4️⃣ Surge.sh (דרך Terminal)
```bash
npm install -g surge
cd recipe-app
surge
```

### 5️⃣ Render (חדש ומומלץ)
1. היכנסי ל-render.com
2. צרי Static Site חדש
3. חברי את ה-GitHub או העלי ידנית
